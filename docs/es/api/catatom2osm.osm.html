append(data, query=None)[fuente]¶

Append data elements to this dataset avoiding duplicates. Optionally filter by query.

Parámetros

  • data (iterable, Osm or Element) – source data

  • query (func) – function to apply to each element and returns a boolean deciding if it will be included or not

Ejemplo

>>> d1 = osm.Osm()
>>> n1 = d1.Node(1, 1, tags={'tourism': 'viewpoint'})
>>> n2 = d1.Node(2, 2, tags={'amenity': 'cafe'})
>>> d2 = osm.Osm()
>>> d2.append(d1)  # all
>>> d3 = osm.Osm()
>>> d3.append(d1, lambda e: 'amenity' in e.tags)  # only amenities
